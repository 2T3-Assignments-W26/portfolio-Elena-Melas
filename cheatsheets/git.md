Git Cheat Sheet

Initializing & Cloning

git init: Initializes a new Git rep in the cd
git clone [with url]: cline a rep from GitHub to device
Staging & Commiting

git status: check status of changes (untracked, modifies or staged)

git add[with filename]: add specific file to git

git remote -v: verifies URL connection
